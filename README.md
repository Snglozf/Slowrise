# Slowrise

Slowrise is a 2D action-platformer game developed using C# and the MonoGame framework. This project has been prepared as the Final Exam Project for the game programming course.

## Developers
* **Büşra Genç** - Programming
* **Songül Özfidan** - Programming

## Game Objective
The primary objective of Slowrise is to progress through various platforms and levels while collecting health pickups along the way to prepare yourself for the ultimate battle. At the very end of the game, the player must face the main enemy (boss) in an epic showdown and defeat them to win the game. If you lose all your health before defeating the boss, the game results in a "Game Over".

## How to Play / Controls
You can control the character using your keyboard:
* **Enter:** Start Game
* **A:** Move Left
* **D:** Move Right
* **W:** Jump
* **Spacebar:** Shoot / Attack
* **R:** Return to Main Menu

## Key Features and Interactive Systems
* **Main Menu:** A standalone title screen independent of the core gameplay, featuring options to start or exit the game.
* **Gameplay Loop:** Level progression, resource management (collecting health), and the final boss battle.
* **Win/Loss Conditions:** Clear winning condition (defeating the boss) and clear losing condition (running out of health).
* **Interactive Systems:** Platform physics (gravity and collision detection), health pickup system, and combat mechanics (shooting and enemy AI).
* **User Interface (UI):** Health bar/text and status indicators to keep track of your survival status.

## Running the Compiled Game
1. Navigate to the **Releases** section on the right side of this GitHub repository.
2. Download the `Slowrise_Compiled.zip` file.
3. Extract the ZIP folder.
4. Double-click the `Slowrise.exe` file to start playing.
